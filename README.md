# Infrastructure

## Fedora Workstation Infrastructure

Automated setup scripts for Fedora KDE Plasma.

### Setiing up a new system

1. Clone the repo 
```bash
git clone https://github.com/fernandodonea/Infrastructure
```

2. Navigate in the setup folder
```bash
cd cd Infrastructure/workstations/fedora-desktop/
```
3. Make the scripts executable
```bash
chmod +x setup.h 
chmod +x /modules/*.sh 
```

4. Run the main script
```bash
./setup.sh
```
